#Features
 1. Built with Django

 3. Charts drawn using [django-nvd3](https://github.com/areski/django-nvd3/blob/develop/django_nvd3/templatetags/nvd3_tags.py) (a Django plugin for [NVD3](http://nvd3.org/) that uses [D3 charts](http://d3js.org/))
